Units of data in a CacheIndex.

Each Archive conatins files, which contain the actual data that can be parsed with the appropriate deserializer in definitions.

None of the structs in this module can be constructed directly. Instead, construct a CacheIndex and use its IntoIterator implementation or its archive method instead.
