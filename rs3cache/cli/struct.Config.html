Fields

input: PathBuf

The path where to look for the current cache.

output: PathBuf

The path where to place output.

render: Vec<Render>
Available on non-WebAssembly only.

This exports them as small tiles, formatted as <layer>/<mapid>/<zoom>/<plane>_<x>_<y>.png, suitable for use with interactive map libraries such as https://leafletjs.com/, as seen on https://mejrs.github.io/

dump: Vec<Dump>

Dumps the given archives.

assert_coherence: bool

Checks whether the cache is in a consistent state. Indices 14, 40, 54, 55 are not necessarily complete.
